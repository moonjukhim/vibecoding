# Lab 

### Lab 목표 : AI 산업 뉴스 조사 → 요약 → 파일 저장

### Lab 시나리오

```text
ai-trend-agent-lab
│
├ agents
│  └ research_agent.py
│
├ prompts
│  └ research_prompt.md
│
├ output
│  └ news_summary.md
│
└ main.py
```

##### Step1 : Agent 환경 구축

```text
Day1에서 설계한 AI Trend Automation 프로젝트를 구현하려고 한다.

다음 조건으로 Python 프로젝트 구조를 생성해줘.

조건

1 Python 기반 프로젝트
2 LangChain과 LangGraph 사용
3 Claude 또는 OpenAI LLM 사용 가능하도록 설계
4 단일 Agent 시스템

프로젝트 이름

agentic-ai

폴더 구조

agents
prompts
output

생성할 파일

main.py
agents/research_agent.py
prompts/research_prompt.md

각 파일의 역할도 설명해줘.
```

##### Step2 : Research Agent Prompt 생성

```text
AI 산업 뉴스 조사 Agent를 만들려고 한다.

Research Agent가 사용할 Prompt를 작성해줘.

조건

1 AI 산업 관련 최신 뉴스 조사
2 주요 기사 요약
3 핵심 트렌드 정리

출력 형식

- 뉴스 제목
- 출처
- 핵심 요약
- 주요 트렌드

이 Prompt를 prompts/research_prompt.md 파일로 생성해줘.
```

##### Step3 : Research Agent 구현

```
Research Agent를 Python으로 구현해줘.

조건

1 LangChain 기반 LLM 호출
2 research_prompt.md를 사용
3 입력

AI 산업 뉴스 조사

4 출력

뉴스 요약

파일

agents/research_agent.py

Agent는 다음 기능을 수행해야 한다.

1 AI 산업 뉴스 조사
2 뉴스 요약
3 결과 반환

코드는 가능한 단순하게 작성해줘.
```

##### Step4 : 메인 프로그램 생성

```text
Research Agent를 실행하는 main.py를 작성해줘.

조건

1 research_agent.py 호출
2 입력

AI 산업 뉴스 조사

3 결과 출력
4 output/news_summary.md 파일로 저장

파일

main.py
```

##### Step5 : 자동 실행 구조

```text
Research Agent가 다음 작업을 자동으로 수행하도록 개선해줘.

자동 작업

1 AI 산업 뉴스 조사
2 뉴스 요약
3 결과 파일 저장

출력 파일

output/news_summary.md

가능하면 코드 구조를 단순하게 유지해줘.
```

##### Step6 : 최종 실행

```text
python main.py
```
