# 1. 제품 개요 (Overview)
제품명

Meeting Copilot (MCP 기반 회의록 자동화 도구)

목적
회의 내용을 자동으로 정리하고,
액션 아이템을 추출하며,
업무 흐름까지 연결되는 AI 기반 회의 관리 시스템 구축

핵심 가치
 - 회의 기록 자동화
 - 중요한 정보 자동 추출
 - 업무 연결 (Action Tracking) 
 - 개인 맞춤 회의 관리


# 2. 문제 정의 (Problem Statement)

현재 문제
- 회의 내용 기록이 번거롭다
- 중요한 내용이 누락된다
- 액션 아이템이 실행되지 않는다
- 회의 이후 업무 연결이 어렵다

# 3. 목표 (Goals)

Primary Goals
- 회의 내용을 자동으로 정리
- 액션 아이템 자동 생성
- MCP 기반 외부 도구 연동

Success Metrics
- 회의록 생성 시간 80% 감소
- 액션 아이템 누락률 감소
- 사용자 만족도 증가
