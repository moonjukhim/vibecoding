# 회의록 작성하기

```text
20개의 서로 다른 회의록 샘플을 만들어줘. 파일명은 "meeting_YYYYMMDD_ID.txt" 형식으로 접근 가능한 파일시스템에 저장해줘.
```

```text
회의록의 주제를 뽑고 주제별 몇 번의 회의가 있었는지 요약해줘.
```

```text
주제별 회의록을 원본은 그대로 두고, 하위에 폴더를 만들어서 폴더별로 분류를 해줘.
```
