### Lab 시나리오

```text
회사에서 다음 업무를 자동화하려고 합니다.

```



```bash
mkdir ai-trend-agent-lab
cd ai-trend-agent-lab
```

##### Step1 : 업무 분석

```text
다음 업무를 자동화하려고 한다.

업무 설명

매주 AI 산업 트렌드를 조사하여
시장 분석 보고서를 작성하는 업무이다.

현재 방식

직원이 직접 검색
→ 자료 정리
→ 분석
→ 보고서 작성

다음 작업을 수행해줘

1 이 업무를 자동화 가능한 단위 작업으로 분해해줘
2 각 작업을 Agent 역할로 정의해줘
3 결과를 Markdown 형식으로 정리해줘
```

```text
위 내용을 docs/task_analysis.md 파일로 저장해줘
```

##### Step2 : 코딩 설계

Promp 기반 workflow 설계

```text
다음 시스템을 설계하려고 한다.

AI 산업 트렌드 분석 자동화 시스템이다.

다음 조건을 만족하는 Prompt Workflow를 작성해줘

조건

1 AI 산업 뉴스 조사
2 주요 트렌드 분석
3 핵심 기업 분석
4 보고서 생성

각 단계별 Prompt를 작성하고
Agent 역할도 정의해줘

결과는 Markdown 문서로 작성해줘
파일 이름은 prompt_design.md로 만들어줘
```

##### Step3 : Agent Workflow 설계

```
AI 산업 트렌드 자동 분석 시스템의 Agent Workflow를 설계해줘.

조건

1 User가 질문을 입력한다
2 Research Agent가 뉴스 조사 수행
3 Analysis Agent가 시장 트렌드 분석
4 Writer Agent가 보고서 작성

다음 내용을 포함해줘

1 전체 Workflow 구조

3 데이터 흐름

결과를 Markdown 파일로 작성해줘
파일 이름은 agent_workflow.md로 만들어줘
```
