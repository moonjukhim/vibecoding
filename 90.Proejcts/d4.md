##### Step1 : Multi-Agent 패턴 이해

```text
현재 AI Trend Automation 프로젝트에는 다음 Agent가 있다.

Research Agent
Analysis Agent
Writer Agent

이 Agent들을 기반으로 Multi-Agent 시스템을 설계하려고 한다.

다음 세 가지 Agent 패턴을 설명하고
각 패턴이 이 프로젝트에 어떻게 적용될 수 있는지 설명해줘.

1 Planner Pattern
2 Manager Pattern
3 Debate Pattern

각 패턴에 대해

- 구조
- 역할
- 장점

을 Markdown 형식으로 작성해줘.
```

##### Step2 : Multi-Agent Workflow 설계

```text
AI 산업 시장 조사 자동화 시스템을 LangGraph 기반 Multi-Agent 구조로 설계해줘.

사용할 Agent

Research Agent
Analysis Agent
Writer Agent

Workflow

Input
 ↓
Research
 ↓
Analysis
 ↓
Report

다음 내용을 포함해줘.

1 Agent 역할 정의
2 데이터 흐름
3 Workflow diagram

결과를 Markdown 문서로 작성해줘.
파일 이름은 multi_agent_workflow.md 로 만들어줘.
```

```text

사용할 Agent

Research Agent
Analysis Agent
Writer Agent

LangGraph Graph 구조

User Input
 ↓
Research Node
 ↓
Analysis Node
 ↓
Writer Node

Python 코드로 구현해줘.

파일

workflow/agent_graph.py

조건

1 LangGraph 사용
2 각 Agent를 Node로 구현
3 상태(State)를 공유하도록 구현
```
